# node-exercise

Clone repo:

```
https://github.com/feegloo/node-exercise.git
```

then install node_modules:

```
npm i
```

## scripts

**run app.js (build + start)**

```
npm run restart
```

**build app.ts**

```
npm run build
```

**start (only after build)**

```
npm start
```

**run app.ts (without build)**

```
npm run ts:node
```

**run unit tests**

```
npm test
```

**run unit tests (in watch mode)**

```
npm test:watch
```
