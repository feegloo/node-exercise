# node-exercise

**run app.js (build + start)**

```
npm run restart
```

**build app.ts**

```
npm run build
```

**start (only after build)**

```
npm start
```

**run app.ts (without build)**

```
npm run ts:node
```

**run unit tests**

```
npm test
```

**run unit tests (in watch mode)**

```
npm test:watch
```
